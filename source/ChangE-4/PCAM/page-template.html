说明

以下PCAM数据基于1月3日发布的2B级PDS文件，共34组2564张。
PCAM可以获取2352×1728（彩色）或1176×864（全色）两种格式的照片。
彩色照片是线性的bayer图像，我的处理脚本debayer后再做了p2p98线性拉伸。
全色照片为机内bin2的灰度图像，脚本只做拉伸。

Use a downloader (such as Motrix) to download all images

NOTE

The following 2564 (34 groups) images are generated from the 2B version PDS files released on Jan 3rd.
The PCAM can produce images in two forms: 2352×1728 (color) or 1176×864 (panchromatic).
For the former, my script debayered and stretched linearly (p2p98) the images before exporting.
The latter were grayscale images that were downscaled internally, so my script only did the stretching.
